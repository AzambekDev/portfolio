Featured Projects

Pneumax

Pneumax is an advanced deep learning-based pneumonia detection system designed to analyze pediatric chest X-rays. Built with Python, TensorFlow, and Keras, it leverages a robust custom Convolutional Neural Network (CNN) and transfer learning via MobileNetV2. To provide medical transparency, the model integrates Grad-CAM (Gradient-weighted Class Activation Mapping) which generates visual heatmaps highlighting the exact lung regions the AI focused on. This ensures clear, evidence-based diagnostic aids alongside the original X-rays.
